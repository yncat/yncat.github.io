ということで、いつものように、１月から順に振り返っていこうと思います。面白かったツイートof the yearは、できればやりたいところですが、最近の私のツイートはべつに面白みがないので、 of the year 対してできないかもしれません。

## １月

前の年からずっとバトルグラウンドにはまっていたようです。で、ちょっと読み上げにバグがあって、 hearthstone access のコードに修正を入れたりしていたようです。

１月からいきなりイベントがありました。東京eスポーツフェスタというイベントで、オーディオゲームセンターとして出展し、私が小学生の時に作った拙作ゲーム「後出しじゃんけん」の展示をさせていただきました。もう一つ、去年のCCBTハッカソンで製作された「キキオン」も展示許可をいただきました。

後出しじゃんけんは、最初の設定だと難しすぎるから、ちょっと速度を落とした状態で展示しようかなと思っていました。が、「この圧倒的に無理な感じと、それでも何回かやってるとできてくる感じがたまらないからこのままのほうがいい」と言われて、ほぼ原作そのままの状態で出すことになりました。非常にありがたいことに、結構すごい評判をいただき、会場にお越しくださった方々も楽しそうにしていた様子でした。

ノートPCに梅酒をこぼしました。ちょっとだけ。ほんとにちょっとだけ。１滴ぐらい。すぐに拭き取ったのに、まだ部屋が暖かいときにキーボードがうにってなります。修理に出したいところではあるのですが、どうせキーボードを基盤ごと交換します１０００００とか言われることが目に見えているので、なんだかなあと思ってそのまま使っています。あかんかなー。

Spider-man 2 を全クリしたらしいです。２週目で動画を収録してYouTubeにアップしていっているのですが、ほぼ最後まで録画しているのに編集してなくて動画がぜんぜん進んでません。ごめんなさい。

副業で、 oAuth の single sign on のクライアント実装をさくっとしていました。専門分野ではありますが、認証認可出身なので、バグらせずにできました。結構助かったと言ってもらえました。

あと、おでかけくんに「法定代理受領通知」という機能を作ってました。同行援護の利用者ごとに、自治体から「いくらお金かかって、そのうち補助が出たのはいくらで、自己負担になったのはいくらやで」ってデータが事業所に送られてくるらしいです。そのExcelを読み込んで、中身をいい感じに解析して、利用者がwebから見られるようにDBに突っ込んで、それを見られる画面も作りました。でも、これは元のデータをまとめて持って来てインポートするまでが結構大変らしく、まだ運用されてないです。たのむー。

これなんですが、元のデータがめちゃくちゃ温かみを感じる結合セル使いまくりExcel方眼紙でありまして。その中から必要なセルを引っ張ってくる作業がですね、それはもうただの職人芸でございました。高校生だったか大学生だったか、それぐらいのときに TCV を作っていましたが、あの頃にツイキャスのhtmlを正規表現だったかただの文字列検索だったかですごい愚直にパースしていたあの時代を思い出しました。

奥多摩会場の Global Game Jam に参加していたのも１月だったらしいです。リズムゲームみたいなのを４人で作りました。チームにいたエンジニアの人に「オレの友達がオタクの会社落ちましたw」って言われたり、サウンドデザイナーの人とプラグインの沼について語り合ったりして、とても楽しい時間を過ごしました。あ、ちゃんとゲームもできましたよ。

本業のほうでは、１月から正式に、今のチームのエンジニアリングマネージャーを引き受けることになりました。そろそろ１年が経つ今にして思うと、まあ今年の私は頑張ったよ。新卒で入った時は、自分がEMをやることになるとは１ミリも考えてなかったよ。

なんか、１月は思った以上に色々やっていたんですね。よく生きてたものだ。

## ３月

このシステムとは別のやつなのですが、本番環境でやらかしまして、心臓ぎゅってなりました。本番環境での作業のためにコンソールに入り、作業を終えました。抜けたと思ってローカル開発に戻って、とりあえず色々変更したしDB resetするかと思ったら、本番コンソール抜けられてませんでした。平たく言うと、本番環境に向かって `bundle ex rake db:drop` という全破壊呪文を唱えました。幸いなことに RAILS_ENV=production の環境だったのでエラーになり、DBが破壊されることはなく無害だったのですが、いやー、わかる人はわかると思いますが、ギュってなりましたね。ギュって。

Claude Code を本格的に使い始めて、並列作業などもするようになりました。そのときに、次のコマンドや指示を入力できるタイミングがなかなかわかりづらいと思ったので、 hook の機能を使って、入力待ちになったタイミングを通知してくれるツールを作りました。これが、初めて 100% Claude Code で製作した作品になります。これ以降、私はどんどん自分でコードを書かなくなり、 Claude Code に全て書かせるようになっていきました。

3月に、会社で結構な高難易度 & 超高速プロジェクトを成功させていたのですが、その功績をアピールしてくださっただれかがいたようで、いわゆる社員表象的な者を受賞しました。光栄ですね！

ちなみに、本人は当日の発表まで聞かされてなかったし、なんなら、「社員表彰カー、そういうの受賞する人はすごいよねー、すごいすごい、あ、デプロイしてねえや」とかいってデプロイ作業だったかなんだったかをしていたら、「catさんおめでとうございます」って言われ初めて、「え？おれなんかやらかした？」とか真顔で言っていました。本人が最後に気づくという。

あと、リリースされてないのですが、おでかけくんの実験的な機能を開発していたり、自分では10000000000%できないメタルギアソリッドデルタというゲームを買って、「自分でできないゲームを8500円で買うってちょっと頭おかしいよねwwwwwwwwwww」とか会社で言いまくったりしていました。

ちなみに、AIがとても優秀になっているおかげで、メニュー画面はほぼ完璧に認識できるようになっています。ただ、ゲーム画面に行くともう無理です。私は、適当に捜査していた結果、ヘイロー降下で地上に舞い降りたスネークを速攻で崖から落っことしてしまい、開始3分で「スネェェェェェェェェェク！！！！！！！！！！！！！！！」となってしまいました。スネークは経験を積んだエリート兵士であるという設定ですが、敵地に潜入して3分で崖から飛び降りしちゃうって、実はミッションやりたくなかったんですかね。人生に絶望していたんですかね。知らんけど。

実は、9月の最初ぐらいに画面共有をしながら一緒にちょっとだけ攻略してもらったんですが、敵兵氏の足音がしないし、線ニューフェイズではしゃべりもしないので、ぜんぜんどこにいるのかわからず、しかも地形が複雑なのでまともに移動できず、お話にならない感じでした。でも、難易度を一番簡単な設定にすると、見つかって銃撃され続けても、5分ぐらいは生きているのでヤバすぎると思います。
